## 杯子取水python逻辑实现
参考[cup-game](https://github.com/albertlee/cup-game)

## 题目：
- 1.现在有两个杯子，一个能装水 6L，一个能装水 5L，不用其他杯子的情况下，怎么倒出 3L 的水？
- 2.现在有两个杯子，一个能装水 6L，一个能装水 5L，不用其他杯子的情况下，怎么倒出 2L 的水？
